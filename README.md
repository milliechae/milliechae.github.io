# 포트폴리오 케이스 스터디 템플릿

## 파일 구성
```
portfolio/
├── index.html   ← 내용 텍스트는 전부 여기서 수정
├── style.css    ← 색상/폰트/여백 등 디자인은 전부 여기서 수정
└── images/      ← 스크린샷, 목업 이미지를 여기에 넣기
```

## 1. 텍스트 수정하기
`index.html`을 열면 `<!-- ① ... -->` 형태의 한글 주석이 각 구역마다 달려 있습니다.
그 부분의 텍스트만 본인 프로젝트 내용으로 바꾸면 됩니다. (HTML 태그는 그대로 두기)

## 2. 이미지 넣기
`images/` 폴더에 아래 파일명으로 이미지를 넣으면 자동으로 반영됩니다.
- `hero.png` — 맨 위 대표 이미지 (권장 1200×800px 이상, 가로형)
- `quote-bg.jpg` — 인용구 배경 사진
- `lock-demo.png` — 다크 배경 목업 이미지
- `hide-demo.png` — 라이트 배경 목업 이미지

파일명을 다르게 넣고 싶다면 `index.html`에서 `src="images/파일명"` 부분만 수정하면 됩니다.
이미지가 아직 없으면 회색 박스 + 안내 문구가 자동으로 보입니다.

## 3. 프로젝트를 여러 개 만들고 싶다면
`index.html`을 복사해서 `project-2.html`처럼 새 파일을 만들고, 안에 내용만 바꾸면 됩니다.
맨 아래 화살표(← →) 링크를 각 파일로 서로 연결해주면 프로젝트 간 이동이 가능합니다.

---

## 4. GitHub Pages로 무료 호스팅하기

1. GitHub에서 새 저장소(repository)를 만듭니다. (예: `my-portfolio`)
2. 이 폴더 안의 `index.html`, `style.css`, `images/` 를 저장소에 업로드합니다.
   - GitHub 웹사이트에서 "Add file → Upload files"로 드래그해서 넣어도 됩니다.
3. 저장소 페이지에서 **Settings → Pages** 로 이동합니다.
4. **Branch**를 `main` (또는 `master`), 폴더는 `/ (root)` 로 설정하고 저장합니다.
5. 몇 분 뒤 `https://<본인깃허브아이디>.github.io/my-portfolio/` 주소로 접속하면 사이트가 보입니다.

> 원하시면 실제 GitHub 계정 저장소 이름을 알려주시면, 제가 파일을 올리는 절차를 
> 명령어(git 기반)로 같이 진행해드릴 수도 있어요. 다만 계정 인증(로그인)은 
> 사용자님 쪽에서 직접 해주셔야 합니다 (제가 대신 로그인할 수는 없어요).
